# Visualization

This folder contains review-oriented rendering utilities for runtime outputs.

## Purpose

Turn computed response data and structural geometry into lightweight figures that can be reviewed without requiring a heavy plotting stack.

## What belongs here

- FRF and mode-shape renderers
- Local HTML/Three.js FEM viewers for BDF geometry and OP2-derived mode shapes
- Export helpers for SVG, CSV, or other review-ready formats
- Plot formatting routines that preserve engineering traceability

## What should not go here

- Numerical response solvers
- File parsers or solver-launch logic
- Knowledge rules or explanatory concepts

## How it interacts with `ava_knowledge`

Visualization presents the evidence generated by runtime calculations. The rendered outputs help engineers assess whether the runtime findings support the rule outcomes defined in `ava_knowledge/`.
