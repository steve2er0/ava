# Index

This folder contains navigation and coverage artifacts for the AVA knowledge base.

## What belongs here

- Knowledge maps, topic indexes, and cross-reference tables
- Coverage views that show which domains have extracts, rules, skills, and tests
- Release-oriented navigation aids for reviewers and agents

## What should not go here

- Primary rule definitions or source extracts
- Standalone engineering content that should live in a domain folder
- Temporary lists that are not maintained with the knowledge base

## How it fits into the AVA pipeline

`index/` is the discovery layer for the system. It lets reviewers and agents find the right knowledge artifact quickly and assess where coverage is complete or incomplete.
