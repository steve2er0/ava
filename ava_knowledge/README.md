# AVA Knowledge System

This directory is the controlled knowledge workspace for the Acoustic and Vibration Agent (AVA). It holds source material, structured extracts, engineering concepts, executable rules, workflow skills, validation tests, and navigation indexes.

## What belongs here

- Versioned engineering knowledge artifacts used by AVA
- Traceable source intake and processed source text
- Rule-ready extracts, concepts, rules, skills, tests, and indexes

## What should not go here

- Temporary solver runs or ad hoc scratch files
- Unreviewed personal notes with no traceable engineering value
- Application code that does not implement or validate AVA knowledge

## How it fits into the AVA pipeline

AVA knowledge flows from `raw/` source intake to `processed/` normalized text, then into `extracts/` for rule-ready statements. `concepts/` stabilize engineering meaning, `rules/` encode deterministic logic, `skills/` apply that logic in workflows, `tests/` verify behavior, and `index/` keeps the system navigable.
